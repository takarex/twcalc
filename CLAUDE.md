# 台灣免費財務計算工具集合站 — Claude Code 專案交接文件

## 專案背景

這個專案由一位台灣開刀房護理師發起，目標是建立一個**零運行成本的台灣財務計算工具網站**，透過 Google AdSense 廣告產生被動收入。

### 核心原則（請務必遵守）
1. **零運行成本** — 純靜態 HTML，所有計算跑在用戶瀏覽器，不需要伺服器、資料庫、API
2. **平民化** — 目標用戶是一般台灣上班族，介面要清楚、不複雜、手機友好
3. **SEO 優先** — 靠 Google 自然搜尋帶流量，廣告收益才能成長
4. **真實數字** — 每個工具都要揭露現有工具（銀行、政府網站）刻意隱藏的資訊

## 六大計算工具規格

### 1. 🏠 房貸試算（進階版）
**核心差異：** 銀行只告訴你月付金，我們還要告訴用戶：
- 寬限期結束後月付跳升多少（代價）
- 提前還款省利息 vs 拿去投資哪個更好（以實際數字比較）
- 總利息佔總還款的比例（讓用戶看清真相）

**輸入欄位：**
- 貸款金額（萬元）、年利率（%）、年限（年）、寬限期（年）
- 每年額外還款（萬元）、手邊資金（萬元）、預期投資報酬率（%）

**輸出：**
- 即時條：每月還款 / 總利息 / 利息佔比
- KPI：寬限期月付、還款期月付、總利息、實際總還款、利息佔比
- 長條圖：本金 vs 利息視覺化
- Insight：寬限期代價 + 提前還款 vs 投資哪個更好的結論

**計算邏輯：**
```javascript
// 本息平均攤還
monthlyPayment = loan * mr * Math.pow(1+mr, pm) / (Math.pow(1+mr, pm) - 1)
// 投資複利
investGain = cash * (Math.pow(1 + investRate, years) - 1)
```

### 2. 💼 跳槽加薪試算
**核心差異：** 帳面加薪不等於實際獲益，要扣掉所有隱性成本

**輸入：** 目前月薪、新月薪、通勤費差異、其他隱性成本、目前年終月數、新年終月數、目前年資、是否失去年資

**輸出：**
- 即時條：每月淨增 / 漲幅 / 損益平衡月數
- KPI：帳面漲幅、每月淨增加、年度淨增、損失年資估值、5年累積、10年累積
- Insight：損益平衡點（月數）+ 綜合建議

**年資損失估算：** `目前月薪 × 年資 × 0.5`（保守估算特休、年資福利等）

---

### 3. 🧾 綜合所得稅試算
**核心差異：** 自動比較標準vs列舉，附稅率級距對照表

**輸入：** 全年薪資、年終獎金、其他所得、扶養人數、身障/重大傷病、人身保險費、健保費、房租、捐贈、醫療費

**115年關鍵數值（每年1月需更新）：**
```
免稅額：97,000 元/人
標準扣除額：131,000 元（單身）/ 262,000 元（已婚）
薪資特別扣除額上限：218,000 元
列舉項目上限：保險 24,000/人、房租 180,000
稅率級距：5% / 12% / 20% / 30% / 40%
```

**輸出：**
- 即時條：應繳稅額 / 有效稅率 / 稅後月收
- KPI：總所得、應稅所得、稅額、有效稅率、稅後年收、稅後月收
- 長條圖：各項扣除的視覺化
- 稅率對照表（高亮顯示適用級距）
- Insight：建議採用哪種扣除方式

---

### 4. 🚗 買車總持有成本
**核心差異：** 揭露折舊是最大隱形成本（佔40–60%），大多數人只看貸款月付

**輸入：** 車價、車型（1600cc/2400cc/電動車）、貸款年數、貸款利率、持有年數、預估轉售價、月油費/電費、月停車費、年保險費、年保養費、年罰單雜費

**車型影響：**
- 1600cc 以下：牌照稅 7,120/年 + 燃料費 4,800/年
- 1601–2400cc：牌照稅 11,230/年 + 燃料費 4,800/年
- 電動車：牌照稅免徵 + 燃料費免徵

**輸出：**
- 即時條：每月真實成本 / N年總花費 / 最大成本項目
- KPI：每月成本 vs 僅貸款月付、N年總花費、折舊損失、貸款利息、日常費用
- 長條圖：折舊 / 日常費用 / 利息的佔比
- Insight：最大成本分析 + 是否建議買中古車

---

### 5. 🌅 退休金缺口計算器
**核心差異：** 含通膨調整，退休時的生活費要換算為當年幣值

**輸入：** 目前年齡、退休年齡、預計壽命、退休後月支出、通膨率、目前存款、每月儲蓄、投資報酬率、勞保月退、其他退休收入

**計算邏輯：**
```javascript
// 退休時資產
futureAssets = savings * (1+r)^years + monthlyContrib * ((1+mr)^months - 1) / mr
// 通膨調整後月支出
adjExpense = expense * (1+inflation)^yearsToRetire
// 每月需多存（複利反推）
moExtra = gap * mr / ((1+mr)^months - 1)
```

**輸出：**
- 即時條：退休時資產 / 缺口 / 每月需多存
- KPI：退休資產、總需求、缺口、每月需多存、退休後調整後月支出、月缺口
- 達成率進度條
- Insight：詳細分析 + 建議

---

### 6. ⚖️ 租屋 vs 買房試算
**核心差異：** 考慮頭期款的機會成本（拿去投資的複利），這是大多數人忽略的

**輸入（買房）：** 總價、頭期款、貸款利率、年限、管理費、預估年漲幅、裝潢費
**輸入（租屋）：** 月租金、年漲幅、頭期款投資報酬率、比較年數

**計算邏輯：**
```javascript
// 買房月支出
moBuyTotal = moMortgage + mgmt + propertyTax/12
// N年後買房淨值
buyNet = futurePrice - remainingLoan - reno
// N年後租屋淨值（頭期款投資）
rentNet = downInvested + extraInvested  // 月付差額也拿去投資
```

**輸出：**
- 即時條（綠色=較優）：買房月支出 / 買房N年淨值 / 租屋N年淨值
- KPI：買房月支出、月租、N年買房淨值、N年租屋淨值、N年後租金、差距
- 兩者比較長條圖（以較高者為100%）
- Insight：結論 + 財務分析 + 非財務提醒

## 重要設計決策記錄

1. **為什麼不用 React/Vue？** 零運行成本原則，純靜態才能免費部署，且無需 build 流程
2. **為什麼不用 Chart.js？** 減少外部依賴，所有圖表用 CSS 長條圖實作，更快更輕
3. **為什麼是單一 HTML？** 部署最簡單，Vercel 一個檔案就能上線，維護者只需管一個檔案
4. **廣告策略：** 財務類廣告 CPM 較高（銀行、保險廣告主），流量累積後廣告收益可觀
5. **網站名稱：** 「台灣免費財務計算工具集合站」SEO 效果最好，直接包含所有搜尋關鍵字
